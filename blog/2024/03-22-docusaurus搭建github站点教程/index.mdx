## 初始化 docusaurus 项目
在这里，我们直接使用 codespaces 进行项目初始化。codespaces 是浏览器版的 vscode，方便我们在线进行服务搭建。

执行命令行：
- `npx create-docusaurus@latest myblog classic`：初始化一个 myblog 的 docusaurus 项目
- `mv ./myblog/* ./myblog/.gitignore .`：将 myblog 目录下的所有文件移动到当前目录

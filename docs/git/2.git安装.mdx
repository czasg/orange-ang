## linux安装

在 linux 中安装 git 比较简单，甚至大部分都是自带的 git 指令。此处主要介绍 ubuntu/centos 系统下 git 安装方法。

```shell title="ubuntu系统"
sudo apt update
sudo apt install git
```

```shell title="centos系统"
sudo yum install git
```

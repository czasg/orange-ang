本教程介绍了 git 的基础原理、常见技巧及使用规范。

在日常工作中，项目每次迭代都会产生新的版本，而通常旧的版本是需要进行保留，保留后的旧版本可以用于历史版本回滚以及存档备份。
因此我们需要一个版本控制管理工具，来帮我们解决这个问题。

git 很好的解决了这个问题。

git 是分布式版本控制系统，常用于源代码及其版本管理。像我们熟知的 github、gitee 等知名站点，就大量使用了 git 作为其代码管理解决方案。从其命名我们也不难看出 git 的含金量。

对于大部分同学而言，只需掌握 git 常用基本指令、了解 git 基础原理，即可满足日常工作需求，进一步深入学习 git 收益反而不大。

因此本教材整体风格会偏向实战型，带大家快速学习并掌握 git 使用。

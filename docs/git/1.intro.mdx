这是面向团队的 Git 教程。

## 背景

目前网上 Git 教程非常多，我找过也看过不少相关的教程，但质量参差不齐。我会将我认为有价值的教程链接放在最后，以便小白同学自行学习。

对于 Git，我其实从未深度学习过，或者说我暂时没有遇到过什么问题，需要我研究 Git 底层原理才能解决。 <br/>
对于 Git，我仅仅是把它当做一个方便的工具使用~

当然，对于一些极端冲突（conflicts）的场景，我也不建议直接使用 Git 命令行去解决，而是应该通过一些成熟的可视化工具去处理。

因此，本教程不会深入探讨 Git 的底层原理和使用方法，而是面向团队的 Git 教程。 <br/>
教程内容将更加注重实践操作，旨在为团队提供 Git 最佳实践指南。


## 目的

- 介绍常用指令
- 介绍团队规范

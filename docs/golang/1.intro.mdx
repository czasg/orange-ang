👏👏👏 这是面向小白的 Go 教程 👏👏👏

Go（Golang）是由 Google 推出的编程语言，目前已被广泛运用于**后端开发、容器虚拟化**等多个领域，可以说是当今最受欢迎的编程语言之一。

相比于其他编程语言，Go 以其**简洁、高效**等优点而备受称赞，并且拥有丰富的标准库和活跃的社区支持。

本教程将深入介绍 Go 的基础原理、常见技巧以及使用规范，旨在帮助开发者快速入门并掌握 Go 语言的核心概念和应用技巧。


## 安装

<BrowserWindow>
    <Tabs>
        <TabItem value="linux-install-go" label="Linux安装Go" default>

1. 确保历史go版本已删除

```go
>>> rm -rf /usr/local/go
```

2. 通过官方渠道下载 [goX.XX.X.linux-amd64.tar.gz](#)，此处以 **go1.22.2.linux-amd64.tar.gz** 为例。

```go
>>> wget https://go.dev/dl/go1.22.2.linux-amd64.tar.gz
...
>>> tar -C /usr/local -xzf go1.22.2.linux-amd64.tar.gz
...
```

3. 添加环境（临时）

```go
>>> export PATH=$PATH:/usr/local/go/bin
```

4. 验证

```go
>>> go version
go version go1.22.2 linux/amd64
```

        </TabItem>
        <TabItem value="windows-install-go" label="Window安装Go">

1. 通过[官方下载地址](https://go.dev/dl/)下载 [goX.XX.X.windows-amd64.msi](#) 并执行安装。此处以 **go1.22.2.windows-amd64.msi** 为例，下载完成后双击即可安装。

2. 安装完成后打开 cmd 命令行输入 `go version` 指令验证是否安装成功。

```go
>>> go version
go version go1.22.2 windows/amd64
```

        </TabItem>
        <TabItem value="goland-install-go" label="Goland安装Go">

1. `File->Settings->Go->GOROOT->+`

2. 安装完成后打开终端命令行输入 `go version` 指令验证是否安装成功。

```go
>>> go version
go version go1.22.2 windows/amd64
```

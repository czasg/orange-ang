## 背景

不同的编程语言通常都具有类似的流程控制结构，但在Go语言中，有一些独特之处。

比如，Go语言没有像其他语言中的 `try ... catch ...` 异常处理机制，而是采用独有的错误链机制... <br/>
再比如，Go语言没有`with lock.lock()`上下文管理机制，而是采用`defer`延迟函数机制... <br/>
...

本节教程主要介绍Go中常用的流程控制机制。

## 目的

学习并了解Go中常用的流程控制机制。

## 条件语句（if）

在 Go 中，`if-else`是常用的条件判断语句，其基本结构如下：

```go
if bool条件 {
  // code
} else if bool条件 {
  // code
} else {
  // code
}
```

## 循环语句（for）

在 Go 中，`for`是常用的循环语句，其基本结构如下：

```go
for 初始值; 条件判断; 循环处理 {
  // code
}
```

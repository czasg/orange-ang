Nginx 是世界上最流行的应用软件之一，几乎每三个网站中就有一个使用了 Nginx。

它的特点是高性能、配置化和支持热部署，可以在不中断服务的情况下对配置进行升级。因此 Nginx 被广泛应用于多种场景下，包括：

- **文件服务器**：对内关联静态文件，对外提供网络文件服务。
- **反向代理服务器**：对内屏蔽真实服务器地址，对外提供请求转发服务。
- **负载均衡服务器**：对外提供统一入口，对内将流量分布到多个服务器上。
- **API网关**：在微服务体系下，配合 Lua 实现限流、熔断等网关功能。

## 目的

本教材和网上其他 nginx 教程会有一定不同。

首先，教程中关于 nginx 的演示都是基于 docker 完成的，几乎不会涉及 nginx进程管理、配置重载等知识。

其次，教程整体会更偏向于实战演练风格，并包含部分核心配置的讲解。


## 安装

请先正确 [安装docker](#)，并执行以下步骤验证 nginx 镜像：

1. 拉取nginx镜像

```bash
$ docker pull nginx
```

2. 查看nginx版本信息

```bash
$ docker run -it --rm --entrypoint=nginx nginx -V
```

```text title="版本信息"
nginx version: nginx/1.21.5
built by gcc 10.2.1 20210110 (Debian 10.2.1-6)
built with OpenSSL 1.1.1k  25 Mar 2021
TLS SNI support enabled
configure arguments: ...
```

如果可以看到类似的输出即表明一切已经准备ok啦~

Nginx 教程

## 安装

本教程中关于 nginx 研发都是基于 docker 容器实现的，请先安装 docker。

```bash
docker pull nginx
```

```bash
docker run -it --rm --entrypoint=nginx nginx -V
```

启动

```bash
docker run -it --rm -p 8080:80 nginx
```
